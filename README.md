# Svelte + Bootstrap Starter

Projeto inicial com Svelte + Vite e exemplo rápido usando Bootstrap (CDN).

Como usar:

1. Instalar dependências:

```bash
npm install
```

2. Rodar em desenvolvimento:

```bash
npm run dev
```

3. Build de produção:

```bash
npm run build
```

Notas:
- Bootstrap está instalado via `npm` e importado em `src/main.js`.
- Se preferir usar CDN em vez de pacote npm, remova as importações em `src/main.js` e adicione o link de CSS no `index.html`.
- Este projeto usa o plugin `@sveltejs/vite-plugin-svelte` — execute `npm install` para baixar as dependências listadas em `package.json`.
